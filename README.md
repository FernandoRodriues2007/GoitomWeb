# GoitomWeb

Site completo para a empresa GOITM, desenvolvido com HTML, JavaScript e TailwindCSS.

## Funcionalidades

- **Página Inicial**: Banner de apresentação da empresa.
- **Seção de Produtos**: Cards interativos para Trigo e Sal.
- **Modal**: Detalhes dos produtos com opção de compra.
- **Integração WhatsApp**: Redirecionamento automático para contato via WhatsApp com mensagem personalizada.
- **Responsivo**: Compatível com celulares, tablets e desktops.
- **Design**: Cores principais amarelo e branco, layout moderno com animações suaves.

## Tecnologias Utilizadas

- HTML5
- JavaScript (ES6)
- TailwindCSS (via CDN)

## Como Executar

Abra o arquivo `index.html` em qualquer navegador web moderno.
